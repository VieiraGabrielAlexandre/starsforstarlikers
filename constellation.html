Configuração da API

Para usar este aplicativo, você precisa configurar suas credenciais da Astronomy API.

Como obter suas credenciais:

  1. Acesse astronomyapi.com
  2. Crie uma conta gratuita
  3. Vá para o painel de controle
  4. Copie sua Application ID e Application Secret
  5. Cole aqui e clique em "Salvar Configuração"
